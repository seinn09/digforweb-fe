# Digital Forensics Management System

This is a code bundle for Digital Forensics Management System. The original project is available at https://www.figma.com/design/wHkOlqYAtEs90fLnOIoI9y/Digital-Forensics-Management-System.

## Setup environment variables

Copy the `.env.example` file to `.env` and fill in the values.

```
cp .env.example .env
```

## Running the code

Run `npm i` to install the dependencies.

Run `npm run dev` to start the development server.
